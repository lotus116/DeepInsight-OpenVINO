# 字体文件夹

此文件夹包含PDF导出功能所需的中文字体文件。

## 字体文件说明

- `NotoSansCJK-Regular.ttf` - 思源黑体，支持中日韩文字
- `SourceHanSansCN-Regular.otf` - 思源黑体中文版
- `SimSun.ttf` - 宋体（备用）

## 使用说明

系统会自动检测并使用此文件夹中的字体文件，无需依赖系统字体路径。

## 字体来源

推荐使用开源字体：
- Google Noto Fonts: https://fonts.google.com/noto
- Adobe Source Han Sans: https://github.com/adobe-fonts/source-han-sans

## 安装字体

1. 下载开源中文字体文件
2. 将字体文件放入此文件夹
3. 重启应用即可自动识别
